# CursorBlur

**An overlay for Windows that draws a motion blur trail behind your mouse cursor.**

# Installation / Usage

Download latest release binary from Releases.

Place .exe anywhere and run, no Administrator permissions required.

## Optional launch arguments:

(Launch args can be prefixed by either / or -)

**sensitivity / s:**  Fade intensity relative to cursor speed.  **Default = 0.03**

**fade / f:**  Time in ms for trail samples to fade out.  **Default = 50.0**

**alpha / a:**  Max opacity of cursor trail.  **Default = 10**

**color / c:**  Tint color of cursor trail.  **Default = #FFFFFF**
